**Weather Data Analysis & Prediction System**

An advanced Machine Learning + Data Analytics dashboard that analyzes historical weather data, compares models, predicts future temperatures, and integrates real-time weather data using a free API.

**Features**
 
 **Data Analysis**

✅ Historical weather data analysis

✅ Feature engineering (date-based features)

✅ Multi-variable dataset (temperature, humidity, wind, rainfall)

 **Machine Learning**

✅ Linear Regression & Random Forest models

✅ Model performance comparison (MAE, MSE, R² Score)

✅ Multi-feature prediction system

 **Interactive Dashboard**

✅ Built with Streamlit

✅ Interactive charts using Plotly

✅ KPI cards (Temperature, Humidity, Rainfall)

✅ City-based filtering

**Real-Time Weather**

✅ Live weather integration using Open-Meteo

✅ No API key required

✅ Real-time temperature & wind speed

**How It Works**

**1. Data Processing**

Cleans dataset

Converts date columns

Extracts features (day, month, year)

**2. Model Training**

Trains multiple ML models

Evaluates performance

Selects best model

**3. Dashboard**

Displays insights & charts

Allows filtering by city

Shows model comparison

**4. Prediction System**

Input → ML Model → Predicted Temperature

**5. Live Weather Integration**

User selects city → API call → Live weather displayed
